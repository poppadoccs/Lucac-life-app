# Lucac LLC — Website Upgrade Research Prompts

10 self-contained research prompts that feed the lucacllc.com website upgrade.
Each prompt is meant to be handed to its **own** independent research agent.

## How to use

1. Open one `prompt-NN-*.md` file.
2. Copy the **entire** file contents into a fresh research agent / chat.
3. Let it run web research and return its findings.
4. Collect all 10 outputs; each ends with the same 6-part summary so they are
   easy to compare and merge into a single website action plan.

Run them in any order — they are independent. Prompts 01–04 shape positioning,
SEO, AI discoverability, and conversion; 05–07 cover backend/security,
performance/accessibility, and analytics; 08–10 cover the chatbot product,
delivery process, and legal/risk.

## Why every prompt repeats the same intro

Each prompt deliberately duplicates the shared blocks (Role, About Lucac LLC,
Current goal, Rules, Canonical pricing, Final output requirements) so it works
standalone with zero extra context. `00-shared-prompt-block.md` holds the single
source of truth for that shared text — edit it there first, then propagate to all
10 files.

## Hard rules baked into every prompt

- Research only — no implementation.
- Cite current sources.
- Every recommendation must become a concrete website artifact: a page, section,
  copy change, metadata/schema, backend/security improvement, tracking event,
  trust signal, risk disclaimer, or checklist item.
- Keep Lucac plain-spoken, trades-friendly, practical — no corporate tone, no AI
  hype.
- No fake proof, fake testimonials, guaranteed leads/rankings, unlimited support,
  or same-day-fix promises.
