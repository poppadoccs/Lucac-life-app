## Research subject

Research the full SEO setup for Lucac. The business is based in/around Tavares,
Central Florida and serves local service businesses and trades.

Include:
- page titles
- meta descriptions
- canonical URLs
- Open Graph image and social preview best practices (dimensions, text, format)
- local SEO for Tavares / Central Florida (NAP consistency, local keywords)
- Google Business Profile connection (how it ties to the site)
- sitemap.xml
- robots.txt
- schema: LocalBusiness, ProfessionalService, Organization, Service, FAQPage,
  BreadcrumbList, and AggregateOffer if appropriate — note exactly which page
  each type belongs on
- whether to add an "Areas Served" page (and what it should contain)
- future resource / blog ideas that fit a trades audience

Assume a multi-page static site with at minimum: Home, Services (and/or
per-service pages), Pricing, About, Contact. Recommend any additional pages.

## Required output

- **Exact title + meta description for every page** — written out, within
  recommended character limits, plain-spoken, no hype.
- **Schema plan** — which schema types go on which pages, with the key fields to
  populate (don't write full JSON-LD, but specify the properties that matter).
- **Sitemap / robots plan** — what belongs in sitemap.xml and what robots.txt
  should allow/disallow.
- **Keyword map** — target keywords mapped to specific pages (primary +
  secondary per page), local + service intent.
- **OG image recommendation** — dimensions, what it should show, text rules.
- **Local SEO checklist** — concrete, ordered, actionable items (NAP, GBP, local
  pages, citations) that map to website changes.
